# ER 图设计（ER Diagram Design）

  1. Please draw an E-R diagram which fulfills the following requirements.
    We want to create a very simple database for MUST in which to record information about professors, students and classes.
    • For each professor we need to store the p-id, name and office number.
    • For each student we need to store the student-id and name.
    • For each class the id (e.g., CS231) and the name.
    • Each class is taught by exactly one professor.
    • Each student must take at least one class.
    • For each class that a student took we need to give the grade.
    • Simplifying assumptions: there is only one lecture for each class and only one semester in the database.
答案

  1. A university registrar’s office maintains data about the following entities:
    • course, including number, title, credits, syllabus, and prerequisites;
    • course offering, including course number, year, semester, section number, instructor(s), timings, and classroom; (course offering depends on the course that university provided)
    • student, including student-id, name, and program;
    • instructor, including identification number, name, department, and title.
    • Further, the enrollment of students in courses and grades awarded to students in each course they are enrolled for must be appropriately modeled.
    • Each instructor must belong to exactly one department.
    • Each department includes dept-id, name, telephone number.
    • Each department must have at least one instructor.
  • Construct an E-R diagram for the registrar’s office. Document all assumptions that you make about the cardinality constraints and participation constraints.
答案

  1. Design a database to organize the information about a factory and the products that are manufactured there. The relevant information is as follows:
    • The factory has a number of employees. For each employee you need to store the name, employee number, and salary.
    • Each employee must be an administrative staff or a worker, but not both.
    • Administrative employees must take seminars. For each seminar we keep its id, name and date. For the administrative staff, you must store the grade received, for each seminar taken.
    • The factory manufactures a number of products and each product is identified by a product id and has a name.
    • Each worker is assigned to work on exactly one product; a product has multiple (one or more) workers assigned to it.
    • A large number of items are manufactured for each product. Each item has a serial number and a color. Different items of the same product have different serial numbers. However, two items that belong to different products may have the same serial number.

# 关系代数（Relational Algebra）

Suppose we have the following relations:

Sailors ( sid, sname, age )
Boats ( bid, bname, color )
Reserves ( sid, bid, date )

Query 1: Find the names of sailors who have reserved boat with bid = 100.

答案

Πsname(σbid=100(Reserves⋈Sailors))\Pi_{sname}(\sigma_{bid=100}(Reserves\Join Sailors))Πsname​(σbid=100​(Reserves⋈Sailors))

Query 2: Find the sid of sailors who have reserved at least a green boat.

答案

Πsid(σcolor=′green′(Boats⋈Reserves))\Pi_{sid}(\sigma_{color='green'}(Boats \Join Reserves))Πsid​(σcolor=′green′​(Boats⋈Reserves))

Query 3: Find the names and age of sailors who have reserved at least a red or a green boats.

答案

Πsname,age(σcolor=′red′(Boats⋈Reserves⋈Sailors))∪Πsname,age(σcolor=′green′(Boats⋈Reserves⋈Sailors))\Pi_{sname,age}(\sigma_{color='red'}(Boats \Join Reserves \Join Sailors)) \cup \Pi_{sname,age}(\sigma_{color='green'}(Boats \Join Reserves \Join Sailors))Πsname,age​(σcolor=′red′​(Boats⋈Reserves⋈Sailors))∪Πsname,age​(σcolor=′green′​(Boats⋈Reserves⋈Sailors))

Query 4: Find the names, sid, and age of sailors who have reserved at least a red boat and at least a green boat.

答案

Πsname,sid,age(σcolor=′red′(Boats⋈Reserves⋈Sailors))∩Πsname,age(σcolor=′green′(Boats⋈Reserves⋈Sailors))\Pi_{sname,sid,age}(\sigma_{color='red'}(Boats \Join Reserves \Join Sailors)) \cap \Pi_{sname,age}(\sigma_{color='green'}(Boats \Join Reserves \Join Sailors))Πsname,sid,age​(σcolor=′red′​(Boats⋈Reserves⋈Sailors))∩Πsname,age​(σcolor=′green′​(Boats⋈Reserves⋈Sailors))

Query 5: Find the names and age of sailors who have reserved all green boats.

答案

Πsname,age(Πsid,bid(Resevers)/Πbid(σcolor=′green′(Boats))⋈Sailors)\Pi_{sname,age}(\Pi_{sid,bid}(Resevers)/\Pi_{bid}(\sigma_{color='green'}(Boats))\Join Sailors)Πsname,age​(Πsid,bid​(Resevers)/Πbid​(σcolor=′green′​(Boats))⋈Sailors)

Query 6: Find the names and age of sailors who have reserved all green boats and all red boats.

答案

Πsname,age(Πsid,bid(Resevers)/Πbid(σcolor=′green′(Boats))⋈Sailors)∩Πsname,age(Πsid,bid(Resevers)/Πbid(σcolor=′red′(Boats))⋈Sailors)\Pi_{sname,age}(\Pi_{sid,bid}(Resevers)/\Pi_{bid}(\sigma_{color='green'}(Boats))\Join Sailors) \cap \Pi_{sname,age}(\Pi_{sid,bid}(Resevers)/\Pi_{bid}(\sigma_{color='red'}(Boats))\Join Sailors)Πsname,age​(Πsid,bid​(Resevers)/Πbid​(σcolor=′green′​(Boats))⋈Sailors)∩Πsname,age​(Πsid,bid​(Resevers)/Πbid​(σcolor=′red′​(Boats))⋈Sailors)

Query 7: Find the names of the oldest sailors.

答案

S1←SailorS2←SailorS3←Sailor−σS1.sid,S1.sname,S1.age(σS1.age<S2.age(S1×S2))Πsname(S3)S_1 \leftarrow Sailor\\ S_2 \leftarrow Sailor\\ S_3 \leftarrow Sailor - \sigma_{S_1.sid,S_1.sname,S_1.age}(\sigma_{S_1.age<S_2.age}(S_1 \times S_2))\\ \Pi_{sname}(S_3)S1​←SailorS2​←SailorS3​←Sailor−σS1​.sid,S1​.sname,S1​.age​(σS1​.age<S2​.age​(S1​×S2​))Πsname​(S3​)
